# Mode

Mode actif : `mixed-en`

- **Code** (classes, fonctions, variables, commentaires inline, docstrings, messages de debug) : en anglais.
- **Docs & prose** (README, CHANGELOG, fichiers de référence, messages de commit, toute documentation) : en anglais.
- **Chaînes destinées à l'utilisateur** (libellés UI, messages CLI, messages d'erreur affichés) : en français.
